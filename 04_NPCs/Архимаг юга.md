# Архимаг юга  *(рабочее имя — TODO)*

**Краткая суть:** один из магов-виновников Падения; выжил, **магократом** правит процветающим южным торговым городом и возглавляет [[Гильдия Навигаторов|Гильдию Навигаторов]], тайно готовя ритуал побега из умирающего мира.

## Кто это
- Происхождение: маг летающего города, причастен к призыву Нечто (см. [[Катастрофа]]).
- Цель: **сбежать из умирающего мира** через ритуал.
- Преимущество: рано разобрался в [[Дрейф реальности|дрейфе реальности]] — потому его город процветает (контролируемая навигация/торговля/укрытие).
- **Власть и рычаг:** архетип «магократия» ([[Формы власти]]); его арканное мастерство дрейфа монетизировано в [[Гильдия Навигаторов|премиальную навигацию]]. [[Союз Искателей]] нанимает Гильдию на срочное → у Архимага **рычаг над мировой торговлей**.

## Роль в кампании
- Опорный момент №1 (см. [[05_Campaign/_Index|Кампания]]). Оформлен как фронт → [[Front — Архимаг юга]].
- Не обязательно «злодей»: его цель понятна, конфликт — в средствах и цене.
- **✅ Побег = вскрыть границу мира** (снимается только изнутри, [[Демоническая война]]); прообраз — маги-завоеватели. Для ритуала собирает **divine-реликвии** (конечный запас, [[Экономика и ресурсы]]) — его сеть скупки через [[Гильдия Навигаторов|Гильдию]] и есть канал, которым он узнаёт об отряде и [[Проводник|Проводнике]].
- **✅ Работодатель в [[Арка 3|Арке 3]]** (уровни 7-10): нанимает партию добыть «якорь» из руин за божественным гейтом. Проводник отнять не может (настройка Темпуса на носителей) — потому нанимает. Дозировка лора — см. [[Арка 3]].
