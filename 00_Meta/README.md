# Homebrew World — волт кампании

Авторский мир для кампании **Pathfinder 2e** (1-20). Жанр — постапокалиптическое фэнтези.

## Как пользоваться
1. Открой папку как **Obsidian Vault** (Open folder as vault).
2. В начале сессии с Claude дай ему `[[claude-instructions]]` (`00_Meta/CLAUDE_instructions.md`).
3. Новые сущности создавай из шаблонов в `00_Meta/Templates/`.
4. Нерешённое — в `[[Open_Questions]]`.

## Структура
- `00_Meta/` — инструкции, шаблоны, открытые вопросы
- `01_World/` — космология, силы, история, метафизика
- `02_Regions/` — регионы и локации
- `03_Factions/` — фракции и организации
- `04_NPCs/` — персонажи
- `05_Campaign/` — сюжет, акты, сессии, таймлайн
- `06_Mechanics/` — хоумрулы и кастомный контент PF2e
- `99_Sketches/` — карты и схемы

## Рекомендуемые плагины Obsidian
- **Templater** — подстановка шаблонов
- **Dataview** — авто-списки по frontmatter (все живые NPC региона X и т.п.)
- **Graph Analysis / локальный граф** — видеть связи мира
