## Космология / метафизика
- [x] ✅ Природа **[[Сердце|Сердца]]** — природный **родник жизни и магии**, питается многомирьем; **с Нечто не связан** (ни родства, ни цикла). Физика — тайна для игроков навсегда.
- [x] ✅ Механизм **сковывания** — боги в цейтнот **бросили себя клеткой-изоляцией** вокруг Нечто в городе; цена — стали узниками затычки и стираются в неё (часы кампании). См. [[Катастрофа]].
- [x] ✅ Связь **Нечто и Сердца** — её нет; роднит лишь место (Нечто в руинах **над** Сердцем).
- [ ] **Дорога домой Нечто** разбита (односторонняя дверь) — что в его мире и просочились ли слуги оттуда? (восстановить/найти дорогу = цель финала)
- [ ] Что со **слугами богов** (сервиторы из душ) после исчезновения хозяев? (кандидаты во фракции) *(NB: это сервиторы богов, не демоны-захватчики из пра-истории — те → [[Культ Пепельной Матери]].)*
- [x] ✅ **Пра-история (до богов):** внешний враг — **демоны из ближнего мира**, вторгшиеся по «дороге демонов». Пятёрка отняла у их лидера **сифон** (демонский инструмент выпивания миров → [[Опустошитель]]), прошла к [[Сердце|Сердцу]], **вознеслась** и уже божественной силой отогнала демонов и **заперла дорогу**. Сердце — родник, не оружие (тема №1, см. [[Архитектурные_приоритеты]]). Истребление прорвавшихся заняло **долгие годы**; горстка демонов уцелела, затаилась и осмелела после Катастрофы → [[Культ Пепельной Матери]]. Осталось: **роль демонов** (минорное участие/отсылка в одной арке; не основная тема) и связь с ancestries (тема №6).
- [x] ✅ **Кап — только на divine-магию, 3-я ячейка**; arcane/primal/occult не тронуты. Остаточная divine идёт **напрямую от ослабших богов** (инстинктивно питают жрецов); угасала постепенно, к лимиту пришла недавно. PC-жрец превышает кап **силой [[Проводник|Проводника]]** (думая, что это бог). См. [[Метафизика магии]], [[Пантеон]] т.2/т.7.

- [x] ✅ Якорь стабильности — **присутствие многих разумных** (консенсус), не буквальный взгляд.

## Гипотезы к проверке
- [x] ✅ **Город стоял над [[Сердце|Сердцем]]** — по вертикали: руины с Нечто на поверхности, **тайное Сердце глубоко под ними**. Игроки находят Сердце поздно и лишь тогда понимают, что оно было прямо под городом. Архимаги не знали, почему линии сходятся здесь.

## Боги
- [x] ✅ Происхождение: пятеро — бывший **отряд приключенцев**, нашедший Сердце. Клирики — после них.
- [x] ✅ Домены пятёрки: **Смерть, Время/Судьба, Жизнь, Справедливость, Удача**.
- [x] ✅ Класс-маппинг = только происхождение богов (НЕ ограничение PC): Время=Wizard, Смерть=Rogue, Жизнь=Druid, Справедливость=Fighter, Удача=Bard.
- [~] **Имена** пятёрки: **Время/Судьба = Темпус** (рабочее; ⚠️ совпадает с FR-богом войны). Остальные четыре — позже.
- [x] ✅ Минимум по всем пятерым заполнен (эдикты/анафема/паства/механика). Осталось: **имена** + поздние секции (сервиторы / карман / сюжет).
- [ ] **Чем боги занимались и каковы их цели** (до Катастрофы и после)? Связано с пра-историей: если их founding-акт — оборона от демонов, это и есть изначальная цель. См. [[Архитектурные_приоритеты]].
- [x] ✅ Модель **посмертия — F**: карманы богов + **Глубина** (старше богов). Сервиторы из душ; дикая нежить из потерянных душ. Детали — [[Загробие]].
- [x] ✅ **Редкость воскрешения**: душу ловят до Глубины; размытая дорога делает это рискованным, PC — особые.
- [x] ✅ Сервиторов делают **все пятеро**; критерий зависит от бога (в нужде — любые души); сервитор **теряет память**.
- [x] ✅ **Глубина (только мастер):** души → чистая жизненная сила → возврат в мир через [[Сердце]] (цикл). Игрокам — полная тайна.
- [x] ✅ Мир умирает **от Нечто** (дрейф/разрыв дороги душ), **не** от убыли жизненной силы. Сердце питают многие миры, доля богов — капля. Грех богов ≠ смерть мира (выбор остаётся трудным).
- [x] ✅ Источник силы Сердца игроки **не узнают никогда** (мастеру: многомирье).
- [ ] Дикая vs созданная нежить: процветают ли некроманты? Меняется ли некромантия в PF2e?

## Кампания (1-20)
- [x] ✅ Ранний Mythic-источник и beat 2 — **разные предметы**. Ранний дар = **[[Проводник]]** (Темпус; капля силы, ведёт героев; ~5 ур.); ключ вознесения = **[[Опустошитель]]** (добывают позже), его заряжают полностью у [[Сердце|Сердца]] в beat 2.
- [x] ✅ Центральный выбор — **матрица 2×2**: цель по Нечто (*уничтожить* / *вернуть домой*) × метод (*восстановить старых богов* / *вознестись самим, боги — жертва*). Нельзя и спасти богов, и вознестись (логика **α**: единственный conduit-артефакт, заряд одноразовый и однонаправленный; старые боги — единственная живая связь с Сердцем, через них либо вливаешь силу в них, либо прожигаешь их собой). См. [[05_Campaign/_Index|Кампанию]].
- [ ] Как показать упавший город рано, не сливая финал? *(частично: руины видны, тайна — Сердце под ними.)*
- [ ] Уровневые вехи угроз (1-4 / 5-10 / 11-16 / 17-20).
- [x] ✅ **Природа артефактов решена** (тема №2). **[[Опустошитель]]** — трофейный **сифон демонов** (в сыром виде выпил бы Сердце; **Темпус доработал** в управляемый conduit); один и невоспроизводим (дорога демонов заперта; пере-настроить нельзя — нужен вымерший эксперт по демоническим артефактам + поверх работы Темпуса) → отсюда же «**почему пятеро**». **[[Проводник]]** — ранний дар Темпуса (Mythic). Осталось — **листья**: имена/формы, как Проводник ведёт, обстоятельства захвата сифона.
- [x] ✅ **Число богов = когорта.** Жёсткой «квоты мест» нет: возносится **группа, вместе прошедшая испытание и дотянувшаяся до Сердца** (основателей было пятеро по составу их отряда). Размер партии финал не ломает. «Почему пятеро» в истории = [[Опустошитель]] **нельзя пере-настроить** под иной/новый пантеон (не квота). ⚠️ *Если хочешь именно жёсткий числовой лимит — скажи, поправлю α.*
