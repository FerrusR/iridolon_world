# Регионы — индекс

> Формы власти по поселениям — [[Формы власти]]. Карта — [[Карта мира]].

- [[Стартовый регион|Везантия]] — **восток, внешнее кольцо**: умеренный речной край (река Маренца) городов-государств
  - [[Сольмаре]] — город-совет в устье; место [[Арка 1|Арки 1]]
  - [[Порто-Кьяро]] — вольный порт-ярмарка (сосед; кандидат Арки 2)
  - [[Лумаредо]] — крепость на перевале к орочьим нагорьям
  - [[Нагорья Везантии]] — дикий горно-лесной фронтир (орки, друиды, ведьмы, оборотни)
- [[Королевство Магнуса]] — держава «всегда-аасимарского» престола; ныне под [[Главный маршал|маршалом]]
- [[Летний Сад]] — локация [[Арка 1|Арки 1]]: сад Жизни, ставший чащей нежити
- [[Тюрьма-остров]] — где заточён [[Кадреш|Отступник]]; стерегут сервиторы [[Справедливость (бог)|Магнуса]]

> **Стартовый город** (город-совет Арки 1) — пока в составе [[Стартовый регион|региона]] / [[Арка 1]]; отдельный файл — по мере имени.
