# Арка 5 — Тюрьма-остров: арка понимания (уровни 13-17)

> ✅ КАРКАС принят (Ferrus, 2026-07-02); детали — листья. Самостоятельная история; из неё партия выходит, **понимая, что такое Нечто** (ядро Z2, [[Пререквизиты финала — чеклист]]).

**Премиса (черновик):** след культа (или нить [[Проводник|Проводника]]) ведёт в океан — к **[[Тюрьма-остров|острову, который всегда на месте]]**. [[Front — Культ Шепчущего|Культ]] идёт **растлить стража-полководца** дряхлеющей тюрьмы (portent 4); партия — за узником, что помнит правду: [[Кадреш|Отступником]].

## Опорные точки
- **Гонка за стража:** оспариваемый сервитор — **полководец [[Смерть (бог)|Мораны]]** среди стражей (кандидат; охотничий корень её ордена — [[Демоническая война]]). Спасён → **благословение Смерти** (№4). Потерян → тюрьма трещит + удар по ключу.
- **[[Кадреш]] — арка понимания:** вторая беседа с демоном за решёткой (рифма с [[Лумаредо]], теперь у партии есть опыт). Даёт Z2-Z5: чуждая природа Нечто, «дороги между миров», печать, вознесение, [[Сердце]], сифон. Дозировать — это ещё не финал.
- **Награда арки — знание о тайнике Темпуса:** где лежит [[Опустошитель]]. **Сама добыча — пролог Арки 6** (чтобы не перегружать эту арку).
- **[[Культ Пепельной Матери|Пепельная Матерь]] — резерв:** её месть Кадрешу канонична, но на сцену выходит, только если партия сама притянет (шум, слухи). См. файл культа.
- **Стражи:** орден «долга без смысла» + дряхлеющие сервиторы/големы — им партия либо союзник против культа, либо взломщик (моральная вилка: тюрьма-то держит их источник правды).
